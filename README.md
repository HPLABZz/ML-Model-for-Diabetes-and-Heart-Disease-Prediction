# 🧬 ML Model for Diabetes and Heart Disease Prediction
## 📌 Project Overview
This project presents a machine learning–based system that predicts the risk of **Diabetes** and **Heart Disease** by integrating **genomic features** with **lifestyle data**. The system is designed to support early diagnosis and enable preventive healthcare through accurate, data-driven insights.

## 🚀 Features
* 🔍 Predicts **Diabetes** and **Heart Disease**
* 🧬 Combines **genomic + lifestyle data**
* 🤖 Uses **Random Forest Classifier**
* 🌐 Web-based interface using **Flask**
* 📄 Generates downloadable **PDF reports (ReportLab)**
* ⚡ Fast and efficient predictions

## 🛠️ Tech Stack
* **Backend:** Python, Flask
* **Machine Learning:** Scikit-learn
* **Data Processing:** Pandas, NumPy
* **PDF Generation:** ReportLab, Pillow
* **Frontend:** HTML, CSS, JavaScript
* **Deployment:** Render

## 📊 Machine Learning Model
* Algorithm: **Random Forest Classifier**
* Handles mixed data (genomic + lifestyle)
* Optimized for classification accuracy and generalization

## 🔄 Workflow
1. User inputs genomic and lifestyle data
2. Data preprocessing and validation
3. Model predicts disease risk
4. Result displayed on web interface
5. PDF report generated for download
